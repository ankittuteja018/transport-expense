Setup guide — Google Sheets in 5 steps
1 Go to sheets.google.com → create a new blank spreadsheet → name it Transport Expenses
2 In the sheet, click Extensions → Apps Script
3 Delete all existing code → paste the entire contents of apps_script.js (provided separately) → click Save
4 Click Deploy → New deployment → Web app → set "Who has access" to Anyone → click Deploy → copy the Web App URL
5 Paste the URL above → click Save → then Sync to Sheet anytime to push all trips to Google Sheets
